  • Member 3 – UI/UX Designer
  • Member 4 – QA / Documentation

Application Overview

TaskTrail helps students organize assignments, track progress, and stay productive. Users can create tasks, set deadlines, and visualize progress using a simple dashboard. The goal is to help students manage academic and personal goals efficiently.

Technology Overview

  • Frontend: HTML5, CSS3, JavaScript
  • Backend: Node.js with Express
  • Database: PostgreSQL or SQLite
  • Hosting: GitHub Pages (frontend), Render or Fly.io (backend)
  • Libraries: Chart.js (visualization), bcrypt.js (password security)
