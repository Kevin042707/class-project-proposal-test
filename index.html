Tower jumper

2D Browser-Based Jumping Game

Creator

Kevin Cordova

Application Overview

Tower Jumper is a simple 2D game where the player controls a character that jumps between platforms to reach higher levels. The goal is to achieve the highest possible score without falling off the platforms. This game demonstrates basic game mechanics, player control, and interactive physics in a browser environment.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Libraries: Optional - p5.js or Phaser.js for game physics
  • Platforms: Browser-based (Windows, MacOS, Linux)
  • Hosting: GitHub Pages

Proposal Last Updated

November 2, 2025

Feature List (Prioritized)

  • Character movement using arrow keys (left and right)
  • Jumping on platforms
  • Score counter based on height achieved
  • Platforms that randomly generate as the player ascends
  • Collision detection to prevent falling through platforms

Future Feature List

  • Power-ups (double jump, speed boost)
  • Moving platforms
  • Background music and sound effects
  • High score leaderboard

Basic Technical Features

  • HTML canvas for game display
  • CSS styling for player and platform visuals
  • JavaScript logic for movement, jumping, and platform generation
  • Simple collision detection

Advanced Technical Features

  • Smooth animations and physics calculations for realistic jumping
  • Dynamic difficulty: platforms get smaller or move faster as score increases
  • Sound and music integration using JavaScript Audio API
  • Persistent high scores using local storage or Firebase
  • Optional game library integration (Phaser.js) for advanced physics
