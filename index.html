Application Overview

CampusConnect helps students find campus events, join clubs, and access helpful resources in one place. It’s designed to make it easier for students to stay connected and informed.

Technology Overview

  • Languages: HTML, CSS, JavaScript
  • Libraries: Optional - Firebase or Chart.js
  • Hosting: GitHub Pages
